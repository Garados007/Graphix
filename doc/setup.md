# Setup

## Development Build

1. Clone GitHub Project
2. Install NuGet Packages
   - SharpDX
   - SharpDX.Desktop
   - SharpDX.Direct2D1
   - SharpDX.Direct3D11
   - SharpDX.DXGI
   - SharpDX.Mathematics
   - SharpDX.XAudio2
3. Build it with Visual Studio

## Runtime Build

1. Install DirectX11
2. Install .NET 4.5.2
3. Get your Product
